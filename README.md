# Wearable Smart Multimeter - PDE 1824 - Group Project

## Overview
 - A compact, wearable multimeter using the Raspberry Pi Pico to measure capacitance and one additional parameter (voltage, current, or resistance). Designed for portability, usability, and real-time feedback.
 - Firmware written in microPy
---

## Features
- Capacitance (555 timer)
- One mode:
  - Voltage (divider)
  - Current (shunt)
  - Resistance (ADC)
- OLED/LCD display
- Button controls
- LED/Buzzer feedback
- Wearable form factor

---

## System
- **Micro Controller:** Raspberry Pi Pico  
- **Inputs:** Buttons, probes  
- **Outputs:** Display, LED, buzzer  
- **Power:** Battery  

---

## Simulation
- Built in Wokwi  
- Includes all modes + UI

## Hardware & Enclosure
- Custom PCB (clean routing, compact layout)
